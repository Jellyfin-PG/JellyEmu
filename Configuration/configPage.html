IGDB Metadata (via Twitch)

Used for game descriptions, release dates, and cover art. Register your app at dev.twitch.tv/console/apps and set the OAuth redirect to http://localhost.

RAWG Metadata

Used as a fallback provider for game metadata and screenshots. Get your free API key at rawg.io/apidocs.

Romm Integration Disabled by default

Connects to your self-hosted Romm instance for metadata, cover art, save-state cloud sync, playtime reporting, collection sync, and screenshot push. Requires Romm v3.4 or later. Authentication uses OAuth2 (bearer token) with Basic Auth as fallback.

Enable Romm integration Enables metadata, cover art, save sync, playtime reporting, collection sync and screenshot push.

Credentials are sent as HTTP Basic Auth on every request to your Romm instance. A secure HTTPS connection is strongly recommended.

Romm Features (requires Romm enabled above)

Save-state sync Pull save from Romm on launch if it's newer; push to Romm after each save.
Playtime reporting Send session playtime to Romm when a game session ends.
Collection sync Import Romm collections as Jellyfin collections.
Screenshot push Send in-game screenshots to Romm when you take one.

Netplay

Optional. Provide your netplay server address to enable multiplayer via EmulatorJS. Leave blank to disable netplay. Requires a running EmulatorJS netplay server.

RetroArch Integration

JellyEmu exposes endpoints so RetroArch can stream ROMs directly from your Jellyfin server. RetroArch does not support importing playlists from a URL — you need to download the .lpl file and place it manually in RetroArch's playlists/ folder.
