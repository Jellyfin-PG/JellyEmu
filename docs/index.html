Installation README

Step 1: Plugin Configuration

Configure the plugin before adding your media to ensure providers are ready.

  • Navigate to Dashboard > Plugins
  • Open the Emulator Library configuration
  • Enter your IGDB Client ID and Secret
  • Enter your RAWG API Key
  • Save settings

Step 2: Library Creation

Create a specific library for your ROM collection.

  • Navigate to Dashboard > Libraries
  • Select "Books" as the Content Type (Required)
  • Add your ROM folder path
  • Enable IGDB and RAWG under Metadata Downloaders
Note: Using the "Books" content type allows JellyEmu to properly inject game metadata into the Jellyfin database structure.
